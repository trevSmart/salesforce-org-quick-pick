# Extension Resources

This folder contains visual assets for the Salesforce Org Quick Pick extension.

## Files

- `icon.png` - Extension icon (128x128px PNG) displayed in VS Code marketplace and extension list

## Icon Requirements

- Format: PNG
- Size: 128x128 pixels (minimum 64x64, maximum 512x512)
- Will be displayed in:
  - VS Code Marketplace
  - Extension manager
  - Extension details
